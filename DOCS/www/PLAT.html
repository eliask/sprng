PLAT variable for make command

SPRNG libraries can be built by executing the following command:

make PLAT=platform

where platform depends on which SPRNG is to be installed. The specific command on the different machines that SPRNG has been tested on are as follows:

    Cray T3D: make PLAT=T3D

    HP/CONVEX Exemplar: make PLAT=CONVEX

    HP Workstation: make PLAT=HP

    IBM SP2: make PLAT=SP2

    PC running Linux: make PLAT=LINUX

    SGI-CRAY Origin 2000: make PLAT=O2K

    SGI Power Challenge: make PLAT=SGI

    SUN running SOLARIS: make PLAT=SOLARIS

    SUN running SUN OS: make PLAT=SUN
